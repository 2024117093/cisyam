实验1：Arduino IDE安装与ESP32 LED闪烁实验

一、实验目的

1. 掌握Arduino IDE的安装与ESP32开发环境配置。

2. 熟悉ESP32开发板的基本使用方法。

3. 学习GPIO输出控制和串口通信的基础编程。

二、实验器材

• 电脑（Windows系统）

• ESP32开发板

• USB数据线

三、实验原理

通过Arduino IDE编写程序，控制ESP32的GPIO引脚输出高低电平，实现LED周期性亮灭；同时通过串口输出调试信息，验证程序运行状态。

四、实验步骤

1. 安装Arduino IDE并配置ESP32开发板支持包。

2. 将ESP32通过USB线连接电脑，安装驱动并选择对应串口。

3. 新建项目，编写LED闪烁和串口打印代码。

4. 编译并上传程序到ESP32开发板。

5. 打开串口监视器，观察LED状态和串口输出信息。

五、实验代码
// 定义板载LED引脚，避免使用"魔法数字"
#define LED_PIN 2

void setup() {
  // 初始化串口通信
  Serial.begin(115200);
  // 初始化板载LED引脚为输出模式
  pinMode(LED_PIN, OUTPUT); 
}

void loop() {
  Serial.println("Hello ESP32!");
  digitalWrite(LED_PIN, HIGH);   // 点亮LED
  delay(1000);                   // 等待1秒
  digitalWrite(LED_PIN, LOW);    // 熄灭LED
  delay(1000);              // 等待1秒
}
六、实验现象与结果

1. ESP32板载LED以1秒为周期循环亮灭。

2. 串口监视器持续输出Hello ESP32!字符串，说明程序运行正常。

3. 开发板电源指示灯常亮，表明供电稳定。

七、实验小结

通过本次实验，成功搭建了ESP32开发环境，实现了LED闪烁和串口打印功能，掌握了GPIO输出控制和基础编程方法，为后续学习ESP32开发打下了基础。
